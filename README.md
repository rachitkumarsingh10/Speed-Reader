A comprehensive web application designed to help you improve your reading speed, pronunciation, and memory retention through interactive training tools. Built with vanilla JavaScript, no frameworks required!

## 🌐 Live Demo

**[Try Speed Reader Now!](https://speedreader2026.netlify.app/)**

## 🌟 Features

### 📖 Reading Trainer
- **Teleprompter-style display** with smooth auto-scrolling
- **Adjustable speed**: 50-500 WPM (Words Per Minute)
- **Customizable font size**: 16-48px
- **Full controls**: Play, pause, restart, stop
- **Fullscreen mode** for distraction-free reading
- **Save & load** custom texts
- **Session tracking** with detailed statistics

### 🗣️ Tongue Twister Trainer
- **Pre-loaded collection** of classic tongue twisters
- **Custom twisters**: Add, edit, and delete your own
- **🔂 Auto-Loop Feature**: Continuously repeat without manual restart
- **Speed control**: 50-500 WPM
- **Font customization**: 20-56px
- **Navigation**: Shuffle and browse twisters
- **Practice tracking**: Repetitions and time
- **Fullscreen support**

### 🧠 Memory Test (NEW!)
- **Test your recall** by reading and writing from memory
- **Customizable timer**: 10-120 seconds reading time
- **Adjustable display**: Font size for comfortable reading
- **Countdown timer** during reading phase
- **Detailed results**:
  - Accuracy percentage
  - Words remembered vs total
  - Side-by-side comparison
  - Analysis of correct, missed, and extra words
- **Retry options**: Same text or new test

### 📊 Statistics Dashboard
- Track total sessions and words read
- Monitor average and fastest speeds
- View detailed session history
- Achievement system
- Export/import data

### ⚙️ Settings & Customization
- Default speed and font preferences
- Auto-save toggle
- Theme selection (light/dark)
- Data management (export/import/reset)

## 🚀 Quick Start

### Installation

```bash
# Clone the repository
git clone https://github.com/rachitkumarsingh10/speed-reader.git

# Navigate to project directory
cd speed-reader

# Open in browser
# Simply open index.html in your preferred browser
# No build process or dependencies required!
```

### Usage

1. **Reading Practice**
   - Navigate to "Reading Trainer"
   - Enter or paste your text
   - Adjust speed and font size
   - Click Play ▶

2. **Tongue Twister Practice**
   - Go to "Tongue Twisters"
   - Select a twister
   - Enable loop mode 🔂 for continuous practice
   - Click Play ▶

3. **Memory Testing**
   - Open "Memory Test"
   - Paste text to memorize
   - Set reading time
   - Start test and read carefully
   - Write what you remember
   - View detailed results
